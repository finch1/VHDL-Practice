Welcome to the Nios II Embedded Design Suite!

The Nios® II Embedded Design Suite (EDS) contains tools required to create Nios II processor hardware systems and write software for them. This page provides resources to accelerate your Nios II design process. For a list of new features, enhancements, and known issues in this release of the Nios II EDS and other Altera products, refer to the Release Notes page on the Altera website.

To get started using the Nios II EDS as quickly and efficiently as possible, follow these steps:

1. Read the Nios II Tutorials

Altera provides tutorials for beginning Nios II developers. The Nios II Hardware Development Tutorial walks you through the process of creating a Nios II processor system, downloading it to the board, and verifying functionality.

The "Getting Started" section in the Getting Started with the Graphical User Interface chapter of the Nios II Software Developer's Handbook contains information on the process of creating and debugging a software project in the Nios II Software Build Tools for Eclipse environment. Refer to this section to get started quickly with software development for the Nios II processor.

2. Refer to Literature and Design Examples

Click the Literature link in the left-hand pane to browse documentation available for the Nios II EDS at the time of this release. For the most up-to-date documentation, visit the Nios II on-line literature page.

The Nios II EDS includes many documented design examples which you can use as the starting point for your own custom Nios II systems. Click the Design Examples link in the left-hand pane for further information.

3. Visit the Nios On-line Community

The Nios Forum, part of the Altera Forum, is an on-line community where you can exchange questions, answers, reference designs, and success stories about Nios II projects. Documentation and designs created by the community are available on the Nios Community Wiki.

4. Get Support and Troubleshooting Answers Online

If you run into trouble, you can get support at Altera's web site.

  1. Refer to the MegaCore IP Library Release Notes and Errata and the Nios II Embedded Design Suite Release Notes and Errata for recently discovered issues with the Nios II processor core and the Nios II EDS. You can find these documents on the Literature: Release Notes page on the Altera website.
  2. Browse the Altera Knowledge Database.
  3. Browse the Nios II Embedded Design Suite Support and SOPC Builder Support pages.
  4. If you still can't find an answer to your question, file a service request on mySupport.
